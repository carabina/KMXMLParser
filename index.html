KMXMLParser

XML Parser for iOS. Get data from an RSS feed in 2 lines of code.

Download this project as a .zip file Download this project as a tar.gz file

KMXMLParser is a simple class you can add to your iOS project to make parsing any standard RSS feed simple. Just drag the class into your project, add a couple of lines of code, and the RSS feed will be returned in an array which allows you to easily access the main elements of the RSS feed.

To parse a feed you only need the following two lines of code:

KMXMLParser *km = [[KMXMLParser alloc] initWithURL:url delegate:self]; NSMutableArray *posts = [km posts];
